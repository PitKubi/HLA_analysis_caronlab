# HLA-I PTM Analysis

Analysis of post-translational modifications on HLA class I presented peptides from the JY cell line.

**Caron Laboratory, Yale University**

## Data

- Cell line: JY (HLA-A*02:01, B*07:02, C*07:02)
- Peptide length: 8-14 amino acids
- PTMs: Phosphorylation, Cysteinylation, Deamidation, Acetylation, Methylation, Dimethylation, Citrullination, Oxidation, Ubiquitination, N-Glycosylation, SUMOylation

## Usage

```bash
Rscript run_all_figures.R
```

## Output

Figures and data tables are saved to `figure_panels/`

Report: `HLA-I_PTM_Analysis_Report.pdf`
